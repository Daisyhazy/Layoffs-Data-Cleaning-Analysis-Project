# Layoffs-Data-Cleaning-Analysis-SQL-Project
A SQL project exploring global tech layoff trends using a public dataset spanning 2020 to 2023.

Cleaning

Removed duplicate records
Standardized inconsistent field formatting
Imputed null values by matching incomplete entries against corresponding records in the dataset
Removed unnecessary rows and columns to improve query performance

Analysis

Identified companies and industries with the highest total layoffs
Tracked monthly trends using rolling totals to surface broader patterns over time
Ranked the top 5 companies by layoffs per year using window functions and partitioning
